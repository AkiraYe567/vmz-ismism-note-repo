* 为什么笔记里有很多英文，看得我好难受

> 这是因为要规避关键字，在`util`目录下有`sensitive_word_replace_english.json`这个替换列表，以及`py_script/word_replace.py`这个替换脚本。
>
> 你可以下载到本地，倒着替换回去看。

* 为什么gitee上顺序错乱？

> 因为他的文件名按字符串ascii排序，而没有对“数字”作优化。您下载到本地就行了，一般本地的操作系统都会便于用户，对文件名中的数字作优化的。
